|removeChild|	删除子节点。|
|replaceChild|	替换子节点。|
|insertBefore(newchild,refchild)|	在指定的子节点前面插入新的子节|点。

|createElement|	创建元素节点。|
|createTextNode|	创建文本节点。|

|setAttribute|	把指定属性设置或修改为指定的值。|
